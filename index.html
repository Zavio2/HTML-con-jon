Etiquetas semánticas estructurales para maquetar el sitio

Div es una estructuras contenedora sin semántica
Header: Cabecera de un sitio web
Main: Se muestra el contenido principal de la página, debe haber uno solo por sitio web
Footer: Pie de página de un sitio web o de una sección
Nav: Para navegar por otros sitios o por diferentes secciones de esta misma página
Article: Se usa para definir una sección de contenido autónoma e independiente
Aside: Representa contenido secundario o complemetario
Section: Representa una sección de contenido genérica, para agrupar y organizar contenido relacionado en bloques
Address: Representa una dirección física o cualquier información de contacto
